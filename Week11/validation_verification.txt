===========================
1. Performance:

5. Reliability:

6. Maintainability:
   - Modular design via Flask Blueprints and Jinja templates; mobile uses React Native. ✔️

7. Portability:
   - Python Flask backend and React Native frontend support iOS and Android. ✔️
